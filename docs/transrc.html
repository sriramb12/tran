NAME

transrc − Transcend .transrc file

SYNOPSIS

PrintByTime: 30 # Legal values 10-120 seconds
PrintBySize: 16 Mb # 1 K - 16 Mb
LogDirectory: . # . | ~ | None
Parallel zip size: 10000000

DESCRIPTION

This is how to create a .transrc file. This file must be placed in the user’s home directory. Default values used by Transcend are set in this file.

BASIC OPTIONS

With the settings given below, Transcend prints out the amount of data transferred every 30 seconds or after the transfer of 16 Mb of data, depending on which event occurs first.

PrintByTime: 30 # Legal values 10-120 seconds

PrintBySize: 16 Mb # 1 K - 16 Mb

The user can set his log directory to the current working directory, his home directory, or no directory.

LogDirectory: . # . | ~ | None

Amount of data to be transcended in parallel and compressed

Parallel zip size: 10000000
